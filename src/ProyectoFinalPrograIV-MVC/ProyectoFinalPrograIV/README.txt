Descripción técnica:
Esta aplicación posibilita la autenticación de usuarios a través de un inicio de sesión seguro, acceso individual a su portafolio y presentación de contenido visual. 
Se empleó arquitectura MVC con Flask y almacenamiento local en SQLite

Requisitos para ejecución:
- Python 3.10+
- pip install flask
- Ejecutar archivo de Python app.py.
